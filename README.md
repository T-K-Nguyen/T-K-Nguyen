### 🛡️ Engineering Feats (Notable Projects)

<img align="left" src="img/hornet2.jpg" alt="Hornet Action" width="300" />

**Hybrid Smart Car Access System**
Architected a secure, offline keyless entry system simulating a vehicle ECU using an ESP32-S3 microcontroller and a mobile app, entirely following the CCC 3.0 standard. Engineered NIST-compliant cryptographic protocols using mbedTLS to execute ECDH, ECDSA, and HMAC-SHA256, thwarting Man-In-The-Middle and Replay attacks. Researched and integrated lightweight TinyML edge models to analyze RSSI signal variance and mitigate physical relay attacks.

**IoT Environmental Monitoring & Control**
Built a concurrent, real-time cabin safety monitor using FreeRTOS Mutexes and Semaphores to achieve an event-driven architecture and eliminate CPU polling. Developed an I2C hot-plug auto-recovery algorithm to prevent bus deadlocks and ensure 100% system uptime during physical sensor disconnections.

**Smart Card Scanning System**
Developed STM32 firmware to interface with smart card readers, optimizing UART communication for reliable data transfer, and integrated an ESP32 module with Wi-Fi to enable remote monitoring.
